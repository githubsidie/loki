# 任务提交功能使用说明

## 功能位置

任务提交功能位于**员工任务详情模态框**中，需要通过以下步骤访问：

### 访问步骤

1. **打开主页面** (`http://localhost:8005`)
2. **查看员工总览区域** - 页面中会显示多个员工卡片
3. **点击任意员工卡片** - 点击后会弹出该员工的任务详情模态框
4. **在模态框底部找到任务提交区域** - 包含以下功能：
   - 任务选择下拉框（显示进行中和待接收任务）
   - 附件上传控件（支持多种文件格式）
   - 完成说明文本框
   - 提交和重置按钮

### 任务提交区域界面

```
任务提交
┌─────────────────────────────────────────┐
│ 选择要提交的任务: [下拉选择框]          │
│ 上传附件: [选择文件按钮] 支持多种格式   │
│ 完成说明: [多行文本框]                  │
│ [完成任务] [重置]                      │
└─────────────────────────────────────────┘
```

7. 查看成功提示，任务状态将更新为"已完成"

### 功能特点

- **任务筛选**: 只显示进行中和待接收的任务供提交
- **文件上传**: 支持Word、Excel、PPT、图片、音乐等多种格式
- **表单验证**: 必须选择任务和输入完成说明
- **状态更新**: 提交成功后任务状态自动更新为"已完成"
- **即时反馈**: 显示成功/错误提示消息

### 注意事项

- 任务提交功能**只在员工任务详情模态框中可用**
- 主页面上的其他区域不会直接显示任务提交功能
- 必须先点击员工卡片才能看到任务提交界面
- 只有状态为"进行中"或"待接收"的任务才能被提交
