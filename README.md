# 任务管理系统 - 测试项目

这是一个简单的任务管理系统，包含完整的用户注册、登录功能和数据持久化存储。

## 功能特点

- ✅ **用户注册**：支持新用户创建账号，包含密码强度检测
- ✅ **用户登录**：安全的登录验证，支持记住用户名
- ✅ **数据持久化**：使用浏览器LocalStorage实现数据本地存储
- ✅ **响应式设计**：适配桌面和移动设备
- ✅ **用户会话管理**：自动检查登录状态，支持安全退出

## 数据存储说明

- 用户数据存储在浏览器的LocalStorage中
- 支持用户数据的增删改查操作
- 当前登录用户信息会被保存并在页面刷新时自动恢复

## 使用指南

1. **注册新账号**：
   - 点击登录页面底部的「注册账号」链接
   - 填写用户名、密码和确认密码
   - 密码需要至少8位，包含字母和数字
   - 点击「注册」按钮完成账号创建

2. **登录系统**：
   - 在登录页面输入用户名和密码
   - 可选择「记住用户名」方便下次登录
   - 点击「登录」按钮进入系统

3. **退出登录**：
   - 在主页右上角点击「退出登录」按钮

## 技术说明

- **前端技术**：HTML5, CSS3, JavaScript (ES6+)
- **数据存储**：LocalStorage API
- **服务器**：Python HTTP Server (用于本地开发)
- **安全特性**：表单验证、密码强度检测、会话管理

## 注意事项

- 本系统仅用于测试和演示，所有数据存储在浏览器本地
- 请勿在生产环境中使用此系统
- 刷新页面不会丢失登录状态
- 清除浏览器缓存可能会导致用户数据丢失
